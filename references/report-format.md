# Report Format

Use this final report shape after producing or verifying HTML.

```md
## Summary

- Output:
- Reference:
- Viewport:
- Result:

## Analysis Notes

- Layout:
- Typography:
- Colors:
- Assets:
- Uncertainties:

## Harness

- Automated score:
- Pixel similarity:
- Diff pixels:
- Accessibility:
- Pass/fail:

## Fidelity Review

- Layout:
- Text:
- Typography:
- Color:
- Spacing:
- Assets:

## Remaining Risks

- Manual text checks:
- Ambiguous source regions:
- Untested viewports:
```

Keep the report short, but include enough evidence that another reviewer can reproduce the decision.
